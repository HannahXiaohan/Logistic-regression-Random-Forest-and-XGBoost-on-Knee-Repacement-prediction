# Conclusion 
Evaluated the Models with AUC( area under curve), P values, and 95% Confidence intervals. The Logistic Regression performed best on this data compared with Random Forest, and XGBoost models. Within the LR output, adding the KL(ordinal) feature improved the baseline's AUC from 0.627 to 0.887. Therefore, we concluded that KL(ordinal) is a significant feature to consider for the knee replacement surgery.
